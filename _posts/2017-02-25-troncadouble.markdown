Scrivi il programma `TroncaDouble` che, data la variabile `double n`, ne stampa a terminale il valore *troncato* a 3 cifre decimali.

Ad esempio per `n =  1.23456789`:

~~~text
> java TroncaDouble
1.234
~~~
